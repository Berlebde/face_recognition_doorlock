# Face Recognition Door Lock with Dahua VTO

Проект для распознавания лиц с видеодомофона Dahua (VTO) и автоматического открытия двери через реле на Raspberry Pi.

## Возможности

- Захват снимка с VTO при нажатии кнопки вызова
- Распознавание лица на снимке
- Сравнение с базой известных лиц
- Открытие замка (реле) при успешном распознавании
- Добавление новых лиц в базу

## Требования

- Raspberry Pi с Raspbian
- Домофон Dahua VTO с IP и учетными данными доступа
- Реле для управления замком (подключено к GPIO)
- Python 3.7+
- Подключение к интернету/локальной сети
